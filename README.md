# react-native-c2c-modules

Native Modules for React Native

## Installation

```sh
npm install react-native-c2c-modules
```

## Usage

```js
import C2CModules from "react-native-c2c-modules";

// ...

const result = await C2CModules.multiply(3, 7);
```
